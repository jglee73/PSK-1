
# Command :  
   SUI__DEBUGGER_D.exe

# Arguments :  -D  (1)  (2)  (3)
   * (1) : goi_name 
   * (2) : class_name
   * (3) : goi dir_root

   Ex)
    -D  JGLEE__IMG_MNG  IMG.ANI  "D:\WORK_PSK\Program_Code\UI_Sample\Debug-GOI"

# Working Directory : 
   SUI__DEBUGGER_D.exe 가 있는 Directory


* SUI Directory 내 _DEBUG 폴더 안에 아래의 파일들이 생성된다.
   _debugger.goi_inf
   _debugger.page
